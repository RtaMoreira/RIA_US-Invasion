Description de notre projet

Par Rita, Vivian et Joao, le 26/02/2019 à 17:18

Dans le cadre du module 624-2, nous développons un jeu en utilisant du HTML, CSS et Javascript. Ce site comprend donc les informations nécessaires à la compréhension du processus de développement de ce jeu.


Le jeu, nommé US Invasion, met en scène la course spatiale vers Mars .

C’est un jeu multijoueur: on y incarne deux compagnies spatiales concurrentes (SpaceX, Blue Origin, Nasa et Roscosmos) qui, à l’aide de leur vaisseau, veulent se rendre sur Mars.

Basée sur le principe d’une course l’un contre l’autre, le joueur qui arrive à mettre suffisamment de distance entre lui et son concurrent gagne la partie (il fait donc sortir le second concurrent de l’écran).

Aussi le voyage deviendra plus ardu au fur et à mesure qu'ils s'enfonceront dans l'espace : les platformes de repos seront en movement ce qui va difficulter leurs atterrissages pour une turbo sieste!


Le code source de l'application est disponible sur le github du projet.
